Mes Projets Académiques

Bienvenue sur mon portfolio. Voici une sélection de projets réalisés durant mon cycle d'ingénieur à l'ENSEM.

Projets de 2ème et 3ème Année

Détection de Panneaux Routiers (Projet Twizzy)

Description : Réalisation d'une application de traitement d'images pour la détection et la reconnaissance en temps réel de panneaux de signalisation.

Compétences : Traitement d'images, OpenCV, Python, Interface graphique.

Classification de Feuilles par Apprentissage Automatique

Description : Construction d'un classifieur de feuilles d'arbres en utilisant des approches probabilistes et des réseaux de neurones, incluant la réduction de dimension (ACP) et l'implémentation de modèles comme les K-moyennes.

Compétences : Machine Learning, Matlab, Analyse en Composantes Principales (ACP), Réseaux de neurones.

Simulateur de Trafic Routier en C

Description : Développement d'un simulateur de trafic en langage C, avec une version concurrente utilisant le multithreading pour gérer les véhicules et les feux de signalisation de manière synchronisée.

Compétences : Langage C, Programmation concurrente (multithreading), Synchronisation (mutex).

Routage et Commutation Réseau (Packet Tracer)

Description : Configuration d'une architecture réseau complète avec Cisco Packet Tracer, couvrant le routage statique et dynamique (RIP, OSPF) et la configuration de commutateurs Ethernet (STP).

Compétences : Réseaux informatiques, Cisco Packet Tracer, Protocoles de routage (RIP, OSPF).

Projets de 1ère Année

Problème du Voyageur de Commerce avec un Robot Lego

Description : Application d'algorithmes d'optimisation pour trouver le plus court chemin pour un robot Lego devant visiter plusieurs points.

Compétences : Mathématiques (optimisation), Automatique (commande de robot), Informatique (C++, Matlab, TCP/IP).

Modélisation et Interrogation d'une Base de Données

Description : Création et gestion d'une base de données d'employés, de la modélisation UML à l'écriture de nombreuses requêtes en SQL.

Compétences : SQL, Modélisation de données (UML), MySQL Workbench.

Calcul d'Itinéraire de Bus avec l'Algorithme de Dijkstra

Description : Développement d'une application en Java pour trouver l'itinéraire le plus rapide dans un réseau de bus en utilisant un graphe et l'algorithme de Dijkstra.

Compétences : Java, Algorithmique (Dijkstra), Structures de données (graphes).
