Mes Projets Académiques

Bienvenue sur mon portfolio. Voici une sélection de projets réalisés durant mon cycle d'ingénieur à l'ENSEM.

Problème du Voyageur de Commerce avec un Robot Lego

Description : Dans ce projet en bureau d'étude, l'objectif était d'appliquer des algorithmes d'optimisation pour trouver le plus court chemin pour un robot Lego devant visiter plusieurs points.

Compétences : Mathématiques (optimisation), Automatique (commande de robot), Informatique (C++, Matlab, TCP/IP).

Modélisation et Interrogation d'une Base de Données

Description : Travaux pratiques sur la création et la gestion d'une base de données d'employés. Le travail a couvert la modélisation (UML, schéma relationnel) et l'écriture de nombreuses requêtes d'interrogation en SQL.

Compétences : SQL, Modélisation de données (UML), MySQL Workbench.

Calcul d'Itinéraire de Bus avec l'Algorithme de Dijkstra

Description : Développement d'une application en Java pour trouver l'itinéraire le plus rapide dans un réseau de bus. Le projet a nécessité de modéliser le réseau sous forme de graphe et d'implémenter l'algorithme de Dijkstra.

Compétences : Java, Algorithmique (Dijkstra), Structures de données (graphes), Interface graphique.
