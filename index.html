Mes Projets Académiques

Bienvenue sur mon portfolio. Voici une sélection de projets réalisés durant mon cycle d'ingénieur à l'ENSEM, en filière Systèmes Numériques (Systèmes Autonomes & IA).

3ème Année

Projet de Fin d'Études : Analyse de Fiabilité des Systèmes

Description : Contribution à un outil logiciel d'analyse de fiabilité basé sur les diagrammes de Hasse pour l'étendre aux systèmes multi-états (au-delà du binaire panne/marche).

Compétences : Sûreté de fonctionnement, Analyse de fiabilité, Diagrammes de Hasse.

Commande Automatique pour l'Atterrissage d'un Avion

Description : Conception d'une loi de commande robuste pour l'atterrissage automatique d'un A320. Linéarisation du modèle, création d'un observateur d'état et réglage d'un contrôleur PID sous Matlab/Simulink.

Compétences : Automatique avancée, Commande de systèmes non-linéaires, Matlab, Simulink.

Pilotage de Systèmes Multi-Agents et Consensus

Description : Simulation et étude de la convergence d'un groupe d'agents autonomes via des algorithmes de consensus en temps continu et discret, basés sur la théorie des graphes.

Compétences : Systèmes multi-agents, Algorithmes de consensus, Théorie du contrôle, Matlab.

2ème Année

Détection de Panneaux Routiers (Projet Twizzy)

Description : Réalisation d'une application de traitement d'images pour la détection et la reconnaissance en temps réel de panneaux de signalisation.

Compétences : Traitement d'images, OpenCV, Python, Interface graphique.

Simulateur de Trafic Routier en C

Description : Développement d'un simulateur de trafic en C, avec une version concurrente utilisant le multithreading pour gérer les véhicules et les feux de manière synchronisée.

Compétences : Langage C, Programmation concurrente (multithreading), Synchronisation (mutex).

1ère Année

Calcul d'Itinéraire de Bus avec l'Algorithme de Dijkstra

Description : Développement d'une application en Java pour trouver l'itinéraire le plus rapide dans un réseau de bus en utilisant un graphe et l'algorithme de Dijkstra.

Compétences : Java, Algorithmique (Dijkstra), Structures de données.

Modélisation et Interrogation d'une Base de Données

Description : Création et gestion d'une base de données d'employés, de la modélisation UML à l'écriture de nombreuses requêtes en SQL.

Compétences : SQL, Modélisation de données (UML), MySQL Workbench.
